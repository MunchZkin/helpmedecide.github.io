# 🎲 Let Fate Decide

A minimal coin-flip decision tool to help you choose between two options — no overthinking, just outcomes.

---

## 🚀 How to Use

1. **Enter your question**
   Example: *“What should I eat?”*

2. **Add two options**
   Example: *Pizza* vs *Burger*

3. **Click “Flip”**
   The coin decides your fate.

4. **View the result**
   One option is chosen — simple as that.

---

## 🧠 Interaction Features

* Coin flip animation for a realistic feel
* Past decisions appear as fading background memories
* Clickable history to revisit previous choices
* Older decisions gradually fade away
* Prevents multiple flips during animation
* Session-based memory (resets when tab closes)
